## :rocket: Instalação:

1. Clone o repositório:
    ```sh
    $ git clone https://github.com/Vitor47/app-firebase.git
    ```
2. Instalar dependências:

    ```sh
    $ cd projectname
    $ npm i
    ```

5. Executando a aplicação:

    ```sh
    $ npm start
    ```
